# ReCore - Documentación del Proyecto

## Descripción
ReCore es una aplicación web full-stack desarrollada con React (frontend) y Spring Boot (backend) con base de datos H2.

## Arquitectura
- **Frontend**: React 19.1.0 + Vite
- **Backend**: Spring Boot 3.5.4 
- **Base de Datos**: H2 (desarrollo) / MySQL (producción)
- **Autenticación**: BCrypt + JWT (pendiente)

## Inicio Rápido
1. **Backend**: Abrir `ReCoreBackend` en IntelliJ y ejecutar `ReCoreApplication.java`
2. **Frontend**: `npm run dev` en la raíz del proyecto
3. **Acceso**: http://localhost:3000 (frontend) | http://localhost:8080 (backend)
